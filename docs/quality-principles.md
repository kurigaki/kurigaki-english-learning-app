# 単語データ品質原則

> 最終更新: 2026-03-31

## 第一原則（最優先事項）

**「そのコースを勉強したら、その試験に合格できる / そのスコアを狙える」**

- 英検○級コースを学習した人は、英検○級に合格できる語彙力を身につける
- TOEIC○○○コースを学習した人は、TOEIC○○○点を狙える語彙力を身につける
- 中学○年コースを学習した人は、中学○年レベルの定期テスト・高校受験に対応できる
- 高校○年コースを学習した人は、高校○年レベルの定期テスト・大学受験に対応できる
- 会話コースを学習した人は、そのレベルの英会話ができるようになる

**この原則に反するデータは、品質テストが通っていても不適切である。**

## 品質基準（優先順）

1. **試験頻出度** — その試験・テストで実際に出題される語であること
2. **レベル適合性** — そのステージの学習者にとって適切な難易度であること
3. **meaningの正確さ** — 試験で問われる意味と一致すること
4. **例文の自然さ** — ネイティブが自然に使う表現であること
5. **アルファベット分布** — 偏りの解消は上記1〜4を満たした上での追加目標

## 禁止事項

- **重箱語の混入禁止**: 試験に出ない専門語・珍しい語をアルファベット偏り解消のために入れてはならない
- **カテゴリ不一致禁止**: TOEICコースに学術語、会話コースにビジネス専門語を入れてはならない
- **レベル超過禁止**: 高校2年コースに英検1級語、TOEIC600コースに900点レベル語を入れてはならない
- **meaning衝突より正確性優先**: meaningが衝突しても、正確な訳を優先する

## データ品質チェック体制

品質会議で以下の4観点からレビューする:

| 観点 | 担当 | チェック内容 |
|------|------|-------------|
| 出題品質 | ヘルパーT | クイズとして適切か、選択肢が混乱しないか |
| カリキュラム | マクロファージ | レベル配置は試験基準に合っているか |
| 記憶定着 | メモリーB | 覚えやすい例文・meaningか |
| 学習者視点 | 一般細胞 | 初学者が見て分かりやすいか |
| データ分析 | 肥満細胞 | 語数・分布・衝突の数値チェック |
| バランス | B細胞 | ゲーム（ダンジョン）で出題バランスが取れるか |
