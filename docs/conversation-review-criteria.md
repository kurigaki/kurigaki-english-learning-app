# 会話コース レビュー基準

> 最終更新: 2026-04-02（CEFR 6段階制への全面再設計時に策定）

## 最優先原則（全コース共通・絶対条件）

以下の3項目は他の全ての観点に優先する。これらを満たさないデータはリリースしてはならない。

### 1. 各コースに出てくる単語のレベルが正しいこと
- A1の語はA1学習者にふさわしい語であること
- B2の語がA2に混入していたり、A1の語がC1に配置されていてはならない
- 判断基準は下記の「CEFR各レベルの定義」に従う

### 2. 頻出単語は全て抑えて、重箱は出題しないこと
- そのレベルの会話で頻出する語が欠けていてはならない（例: A1にbathroom/jobがない）
- 日常会話で使わない専門語・学術語を含めてはならない（例: B2にapartheid/census）
- 「語数を増やすために質を下げる」は絶対禁止

### 3. meaningが正しいこと
- 学習者が間違った意味を覚えることは絶対に避ける
- meaning は10文字以内の正確な日本語訳であること
- 同一ステージ内でのmeaning衝突がないこと

**上記3項目を達成してから、以下の観点（クイズ品質・ダンジョンバランス等）を検討する。**

## CEFR各レベルの定義と語の配置基準

### A1 旅行英会話（~300語）

**できること**: 自己紹介、簡単な質問への応答。相手がゆっくり話せば対応可能。

**ふさわしい語の特徴**:
- 旅行者が最低限必要な語（挨拶・買い物・道案内・食事注文）
- 非常に高頻度で短い語（go, eat, big, red）
- 具体的で目に見えるもの（house, apple, dog）

**含めるべき**: 基本動詞（go, come, eat等）、家族（mother, father, husband, wife）、食べ物、身体（head, hand, eye）、色、数字、曜日・月名、基本形容詞（good, bad, big, small）

**含めてはいけない**: 抽象概念、複雑な文法を要する語（already, enough）、専門的な身体部位（shoulder, elbow）

### A2 日常英会話（~450語）

**できること**: 日常的なやり取り。買い物・食事注文・方向案内。身近な話題で短い会話。

**ふさわしい語の特徴**:
- 居住者の日常生活で使う語（A1の旅行者から一段上）
- 日常の動作・感情・物品
- 基本的な意見表現の入口

**含めるべき**: 日常動詞（believe, choose, fix）、衣類、食材、感情（afraid, angry, bored）、天気、間違い（mistake）、自転車（bicycle）

**含めてはいけない**: 分析・議論用の動詞（compare, explain, suggest → B1）、抽象的な形容詞（independent, powerful → B1）、フォーマルな副詞（obviously, unfortunately → B1）

### B1 社交英会話（~460語）

**できること**: 意見を述べ、理由を説明できる。経験を語れる。旅行中の大抵の状況に対処可能。

**ふさわしい語の特徴**:
- 意見表明・社交・ニュース話題で使う語
- 基本的な句動詞（find out, give up, end up）
- NGSL-Spoken語（stuff, guy, awesome, basically, kind of）
- 抽象的だが日常的に使う名詞（opportunity, relationship, experience）

**含めるべき**: 意見動詞（argue, claim, agree, disagree）、基本句動詞、感情の深堀り（frustrated, overwhelmed）、社会語彙（education, environment, economy）

**含めてはいけない**: 分析系動詞（analyze, evaluate, estimate → B2）、アカデミックレジスターの形容詞（considerable, significant → B2）、フォーマル接続詞（hence, thus, furthermore → B2）

### B2 ディスカッション（~450語）

**できること**: ネイティブと自然に議論。詳細な説明。抽象的なトピックで意見の攻防。

**ふさわしい語の特徴**:
- 抽象的な議論・ビジネス・ニュアンスのある表現
- フォーマルな接続表現（nevertheless, furthermore, in terms of）
- 分析・評価の語彙（assess, evaluate, imply）
- ビジネス語彙（accountability, stakeholder, revenue）

**含めるべき**: 議論動詞（justify, contradict, negotiate）、抽象名詞（implication, paradox, perspective）、フォーマル接続表現、ビジネス語彙

**含めてはいけない**: 政治専門用語（apartheid, coup, embargo → C1）、日常で使わない感情語（contemptuous, indignant → C1）、会話で使わない学術語（empirical, configuration → C1）

### C1 ネイティブ表現（~320語）

**できること**: 流暢かつ自在に表現。フォーマル/カジュアルの使い分け。ニュアンス・ユーモア。

**ふさわしい語の特徴**:
- イディオム（beat around the bush, cost an arm and a leg）
- 高度な句動詞（非自明な意味を持つもの: boil down to, stem from）
- スラング・口語（sketchy, stoked, laid-back）
- 反応表現（I beg to differ, couldn't agree more）
- レジスター切替語（commence=start, reside=live）

**含めるべき**: 非自明なイディオム、高度な句動詞、スラング、婉曲表現（passed away, between jobs）、文化的に一般化した引用、ことわざ

**含めてはいけない**: 基本的な句動詞（find out, give up, pick up → B1）、日常的な反応表現（no way, fair enough, it depends → B1）、文学専用語（mellifluous, perspicacious → C2）

### C2 達人表現（~160語）

**できること**: 微妙な意味の違いを使い分け。ネイティブ同等。比喩・皮肉・文化的表現。

**ふさわしい語の特徴**:
- 文学的・修辞的な語（sardonic, ephemeral, whimsical）
- ギリシャ・ローマの文化的引用（Achilles' heel, Pyrrhic victory, Pandora's box）
- 高度な婉曲・控えめ表現（economical with the truth, to put it diplomatically）
- 法律・学術レジスターの語（pursuant to, vis-à-vis, aforementioned）
- ことわざ（能動的に引用できるレベル）

**含めるべき**: ネイティブでも日常会話では使わない高度な語、文化的教養を示す表現

**含めてはいけない**: 一般化したイディオム（go down the rabbit hole → C1）、よく知られたことわざ（practice makes perfect → B1-C1）

## レビュー観点（最優先原則を満たした上で確認）

### 観点1: CEFRレベル適合性（カリキュラム設計）

各語が正しいCEFRレベルに配置されているか。

**チェック方法**:
- 「この語をA2の学習者が知っている必要があるか？」
- 「この語はB1の社交会話で使うか？それともB2の抽象議論か？」
- 上記の各レベル定義に照らして判断

**よくある問題パターン**:
| パターン | 例 | 対処 |
|---------|-----|------|
| 基本句動詞がC1にある | find out, give up, pick up | B1に移動 |
| フォーマル接続詞がB1にある | hence, thus, furthermore | B2に移動 |
| 政治専門語がB2にある | apartheid, coup, embargo | C1に移動 |
| 日常的な反応表現がC1にある | no way, fair enough, it depends | B1に移動 |
| 一般化したイディオムがC2にある | go down the rabbit hole | C1に移動 |
| 自己紹介語がA1にない | job, country | A1に追加 |
| 旅行必須語がA1にない | bathroom, toilet | A1に追加 |

### 観点2: クイズ出題品質（問題作成）

4択クイズとして成立するか。

**チェック項目**:
- 各品詞に最低20語あるか（4択の組み合わせに十分なバリエーション）
- 品詞の偏りがないか（名詞50%以上は偏りすぎ）
- meaning衝突（同じステージ内で同じmeaningの語）がないか
- `word+partOfSpeech`の重複がコース内にないか

**品詞バランスの目安**:
- 名詞: 30-40%
- 動詞: 20-30%
- 形容詞: 20-25%
- 副詞: 5-10%
- other: 5-15%（B1以上はフレーズが増えるため高くなる）

### 観点3: ユーザー体験（UXテスト）

学習者がそのステージを選んだ時の体験として自然か。

**チェック項目**:
- 「A1を選んだ初心者が`tongue`を学ぶのは違和感がないか？」
- 「B2を選んだ中上級者が`census`を学ぶ必要があるか？」
- displayNameから期待されるコンテンツと一致しているか
- ステージ間の難易度の飛躍が大きすぎないか

### 観点4: ダンジョンバランス（ゲームバランス）

WORD DUNGEONのクイズとして機能するか。

**チェック項目**:
- 1フロア20問×想定フロア数に対して語数は十分か
- 同じ語が毎フロア出現する「デジャヴ感」がないか
- other品詞（フレーズ）が多いステージでは4択のダミー選択肢が足りるか

**最低ライン**: 品詞あたり20語以上

### 観点5: データ整合性（テスト自動検証）

自動テスト（85本）で検証される項目:
- `meaning` ≤ 10文字
- `examples` が正確に3件
- `partOfSpeech` が有効値（noun/verb/adjective/adverb/other）
- `word+partOfSpeech` の重複なし
- コース間のID衝突なし
- meaning衝突が上限以下

## 過去のレビューで発覚した典型的な問題

### 問題1: エージェント独立生成による重複

**状況**: 6つのCEFRレベルを6エージェントが独立に生成した結果、312語がステージ間で重複した。

**原因**: 各エージェントが他ステージの語を知らずに選定したため、`suggest`, `opinion`, `confused`等の複数レベルにまたがる語が重複。

**教訓**:
- 下位ステージに残し、上位から削除する（学習者が最初に出会うレベルに配置）
- エージェント間で「使用禁止語リスト」を共有すべき

### 問題2: 句動詞のレベル配置

**状況**: `find out`, `give up`, `pick up`等の基本句動詞がC1（ネイティブ表現）に配置された。

**原因**: 句動詞は「英語学習者が苦手な表現」として上級に分類されがちだが、会話コースでは日常的に使う頻度が高い。

**教訓**:
- **基本句動詞**（自明な意味: find out=見つける, give up=あきらめる）→ B1
- **高度句動詞**（非自明な意味: boil down to=要するに, stem from=に起因する）→ C1

### 問題3: フォーマル接続詞のレベル

**状況**: `hence`, `thus`, `furthermore`, `moreover`がB1に配置された。

**原因**: 「接続表現」として一括りにされ、`however`や`although`と同レベルに配置された。

**教訓**:
- B1の接続表現: however, although, in addition, on the other hand
- B2の接続表現: nevertheless, furthermore, hence, thus, consequently

### 問題4: 婉曲表現のレベル配置

**状況**: `passed away`, `let go`, `not exactly`がC2→B1に降りてきた。

**原因**: C2から「易しすぎる」として降格されたが、降格先の判断が難しかった。

**教訓**:
- `not exactly`, `could be better` → A2（日常表現として非常に基本的）
- `passed away` → C1（婉曲表現として教育的価値がある）
- `let go` → A2（基本的な意味「手放す」として）

## 学術的根拠

### 語彙カバー率の研究

| 根拠 | 数値 | 出典 |
|------|------|------|
| 会話英語の95%カバー | ~2,000-3,000語族 | Nation (2006) |
| 会話英語の98%カバー | ~5,000-6,000語族 | Nation (2006) |
| NGSL（一般英語） | 2,818語で92%カバー | Browne et al. (2013) |
| NGSL-Spoken（会話特化） | 721語 | Browne & Culligan (2016) |
| English Vocabulary Profile | A1-C2合計 約7,000項目 | Cambridge |

### 現在の会話コースの位置づけ

合計2,163語は、学術基準の2,000-3,000語族の範囲内。会話英語の95%カバーに近い水準。

### NGSL-Spokenの活用

NGSL-Spokenの721語は「会話では高頻度だが書き言葉には少ない語」。これらは主にB1レベルに配置すべき（stuff, guy, awesome, basically, kind of, sort of等）。A2には一部の非常に基本的なもの、B2以上には含めない。

### English Vocabulary Profile (EVP)との整合

EVPは同じ語でも意味によってCEFRレベルが異なる（例: `run` A1=走る, B1=経営する）。将来的にEVPとの照合を行い、語ごとのレベル精度を上げることを推奨。
